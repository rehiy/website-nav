# 开发者导航

实用开发者工具导航，包含各类开发者工具网址，所有网址收录文件在 `/src/website` 下存放。

演示站点：https://nav.tdp.fan

## 快速启动

```sh
npm i
npm run dev
```

## 快速构建

```sh
npm run build
```

## 启用 GraphQL 后端

- 参考 `/src/helper/graphql.ts` 请求语句，实现后端查询支持

- 修改文件 `/src/app.vue` 中的 `/helper/jsondata` 为 `/helper/graphql`
